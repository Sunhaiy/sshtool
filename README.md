<a name="english"></a>
## 🇬🇧 English

**SSHTool** is a modern, lightweight, and cross-platform SSH client built with Electron, React, and TypeScript. It offers a sleek user interface and powerful features to manage your remote server connections efficiently.

### ✨ Features

*   **Server Management**: Easily add, edit, and manage multiple SSH connections.
*   **Terminal Emulator**: Fully functional terminal based on Xterm.js with support for resizing and colors.
*   **System Monitoring**: Real-time dashboard showing CPU usage, Memory, Disk, and Network traffic of your remote server.
*   **File Manager**: Built-in SFTP client to browse, upload, download, and manage files.
*   **Modern UI**: Clean, dark-themed interface designed for developers.
*   **Cross-Platform**: Windows, macOS, and Linux support (via Electron).

### 🚀 Getting Started

#### Prerequisites
*   Node.js (v18 or higher)
*   npm or yarn

#### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/sshtool.git
    cd sshtool
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start development server:
    ```bash
    npm run dev
    ```

4.  Build for production:
    ```bash
    npm run dist
    ```

### 🛠️ Tech Stack

*   **Electron**: Desktop application framework
*   **React**: UI library
*   **TypeScript**: Type safety
*   **Vite**: Fast build tool
*   **Tailwind CSS**: Styling
*   **ssh2**: SSH client implementation
*   **Xterm.js**: Terminal component
*   **Recharts**: Data visualization

## 🇨🇳 简体中文

**SSHTool** 是一款基于 Electron、React 和 TypeScript 构建的现代、轻量级跨平台 SSH 客户端。它提供了简洁的用户界面和强大的功能，帮助您高效地管理远程服务器连接。

### ✨ 功能特性

*   **服务器管理**: 轻松添加、编辑和管理多个 SSH 连接。
*   **终端模拟器**: 基于 Xterm.js 的全功能终端，支持窗口调整和色彩显示。
*   **系统监控**: 实时仪表盘，展示远程服务器的 CPU 使用率、内存、磁盘和网络流量。
*   **文件管理**: 内置 SFTP 客户端，支持浏览、上传、下载和管理文件。
*   **现代 UI**: 专为开发者设计的简洁暗色主题界面。
*   **跨平台**: 支持 Windows、macOS 和 Linux（基于 Electron）。

### 🚀 快速开始

#### 环境要求
*   Node.js (v18 或更高版本)
*   npm 或 yarn

#### 安装步骤

1.  克隆仓库:
    ```bash
    git clone https://github.com/yourusername/sshtool.git
    cd sshtool
    ```

2.  安装依赖:
    ```bash
    npm install
    ```

3.  启动开发服务器:
    ```bash
    npm run dev
    ```

4.  构建生产版本:
    ```bash
    npm run dist
    ```
